# Teste Técnico – Analista de Suporte à Aplicação

Este repositório contém a resolução de um teste técnico para a vaga de **Analista de Suporte à Aplicação**, com foco em:

* Priorização de chamados
* Análise e resolução de incidentes
* SQL (MySQL 5.6+)
* Conceitos de programação e HTTP/HTTPS

O objetivo deste projeto é demonstrar meu raciocínio técnico, capacidade de análise, organização e comunicação em cenários reais de suporte à aplicação.
