# Teste Técnico – Analista de Suporte à Aplicação

Este repositório contém a resolução de um teste técnico para a vaga de **Analista de Suporte à Aplicação**.

- **Priorização de chamados** (Round I)
- **Análise e resolução de incidentes** (Round II)
- **SQL (MySQL 5.6+)** (Round III)
- **Conceitos de programação e HTTP/HTTPS** (Round IV)

O objetivo deste projeto é demonstrar meu raciocínio técnico, capacidade de análise, organização e comunicação em cenários reais de suporte à aplicação.
